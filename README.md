# PingMe Messenger - Full Stack

A complete real-time messenger application with frontend (JavaFX) and backend (Java Socket Server + SQLite).

## Folder Structure

- `frontend/` → JavaFX application (signup, login, forgot password UI, chat interface)
- `backend/`  → Java server (socket-based, authentication, SQLite database)

## How to Run

### 1. Start the Backend Server

```bash
cd backend
java com.codes.server.Server
